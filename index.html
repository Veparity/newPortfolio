BugTracker MVC

Zapp'd

In my application, I have developed a role-based bug tracker that mirrors a widely used tool across different industries. This comprehensive tool allows you to create a Company, assume the role of an Admin, invite Submitters who can then turn into Project Managers to oversee projects, or create Developer users to oversee tickets created for the projects. It enables you to efficiently track the progress of tickets from the moment they are opened until completion, and with teamwork, you can easily eradicate any bugs.

In a duration of three weeks, I accomplished a project encompassing the front-end, back-end, and security measures. The project, built on Railway, was developed using C# and .Net, and incorporates HTML, Javascript, Bootstrap, and PostgreSQL, all organized in a clear MVC design pattern.
